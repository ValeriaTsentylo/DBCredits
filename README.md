# DBCredits

## Overview

DBCredits is a Python project that demonstrates importing and managing structured data from CSV files using SQLite.

The project simulates a credit management system where information about users, plans, credits, and payments is loaded into a relational database.

## Features

- CSV data import
- SQLite database integration
- Relational data modeling
- Data validation
- Basic ETL workflow

## Data Sources

- Users.csv
- Plans.csv
- Credits.csv
- Payments.csv

## Technologies

- Python
- SQLite
- CSV

## Skills Demonstrated

- Data modeling
- Database design
- ETL processes
- Data loading
- SQL fundamentals
- Working with structured datasets

## Project Structure

- CSV files as source data
- SQLite database as storage
- Python scripts for data processing
